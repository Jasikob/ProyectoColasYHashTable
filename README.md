# Proyecto Colas de Prioridad y Hash Tables
Integrantes:

Jesús Schneider 29626956 jesus.schneider@correo.unimet.edu.ve

Marco Alcantarini 30612256 marco.alcantarini@correo.unimet.edu.ve

Este proyecto implementa un simulador de cola de impresión utilizando un montículo binario (min-heap) como estructura de datos principal para gestionar las prioridades. El sistema permite modelar el comportamiento de un sistema operativo en la administración de trabajos de impresión, donde los documentos pueden tener diferentes niveles de prioridad según el tipo de usuario que los envía.
